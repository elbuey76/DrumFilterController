# Spécification fonctionnelle

Ce dossier contient la spécification du contrôleur de filtre à tambour. Le format principal est Markdown afin de mélanger facilement texte, tableaux, images, schémas Mermaid et liens vers des fichiers externes.

## Comment ajouter des visuels

- Placer les images, captures, croquis ou photos dans `docs/specification/assets/`.
- Référencer une image avec la syntaxe Markdown :

```markdown
![Description du visuel](assets/nom-du-fichier.png)
```

- Utiliser des blocs Mermaid pour les schémas simples directement dans le texte.
