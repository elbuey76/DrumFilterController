# Journal de décisions

Ce dossier contient les décisions structurantes du projet. Chaque decision importante doit être ajoutée sous forme d'ADR afin de conserver le contexte et les raisons du choix.

## Convention de nommage

```text
ADR-0001-titre-court.md
ADR-0002-titre-court.md
```

## Statuts possibles

- Proposee
- Acceptée
- Remplacée
- Abandonnee

## Modèle rapide

Copier le fichier `_template.md` pour créer une nouvelle decision.
