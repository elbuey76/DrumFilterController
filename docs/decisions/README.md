# Journal de decisions

Ce dossier contient les decisions structurantes du projet. Chaque decision importante doit etre ajoutee sous forme d'ADR afin de conserver le contexte et les raisons du choix.

## Convention de nommage

```text
ADR-0001-titre-court.md
ADR-0002-titre-court.md
```

## Statuts possibles

- Proposee
- Acceptee
- Remplacee
- Abandonnee

## Modele rapide

Copier le fichier `_template.md` pour creer une nouvelle decision.
