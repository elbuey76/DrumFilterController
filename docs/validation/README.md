# Validation

Ce dossier contient le plan de tests et les preuves de validation du controleur de filtre a tambour.

Le principe retenu est une validation tracee exigence par exigence : chaque exigence fonctionnelle ou de securite doit etre couverte par un test, une inspection ou une justification explicite de report.

Fichier principal :

- [Plan de tests par exigences](plan-tests-exigences.md)

Les procedures longues ou risquees peuvent etre detaillees dans des fiches de test separees, tout en restant referencees par la matrice de tracabilite.
